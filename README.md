# Madina Fish Feed Mill - Static Website

**Official website for Madina Fish Feed Mill, Trishal, Mymensingh, Bangladesh**

## Live Website
**URL:** https://madinafishfeedmill.com

## Pages
- **Home** - Hero, About, Products, Stats, Testimonials, CTA, Contact
- **About** - Company story, Mission/Vision, Factory, Quality
- **Products** - 6 product cards with WhatsApp order buttons
- **Process** - 6-step manufacturing timeline
- **Gallery** - Photo gallery with lightbox
- **Blog** - Articles with categories and newsletter
- **Contact** - Form, Google Maps, WhatsApp/Call/Email CTAs
- **Privacy Policy** - Full legal template
- **Terms & Conditions** - Complete terms

## Features
- Fully responsive (mobile, tablet, desktop)

- Premium green & gold design
- WhatsApp floating button
- Lightbox gallery
- SEO optimized (meta tags, Open Graph, Schema.org)
